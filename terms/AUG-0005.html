Integrated Operator [INO]

Term code: AUG-0005 · Discipline: Agentic AI (Verification) (AV) · Period: 3 · Language: de

Description

Beobachtbarer Arbeitsstil: Integration von KI in Denk- und Arbeitsprozesse mit fliessenden Uebergaengen.

Operational definition

An Integrated Operator is observed when a participant blends AI-assistance fluidly into a primary task such that AI-mediated steps are not separable into distinct sub-tasks, evidenced by uninterrupted task-flow telemetry and post-task think-aloud reports treating AI as embedded rather than tool-like.

Measurement schema

Task-flow telemetry: mean time between AI-handoff and task-resumption (target < 8s); think-aloud coding for AI-as-tool vs AI-as-extension framing on a 5-point integration scale. Composite integration-index = (1 - normalized handoff-time) x integration-scale-mean.

Example data

{
  "task_id": "T029",
  "participant": "P014",
  "ai_handoffs": 22,
  "mean_handoff_time_s": 5.4,
  "integration_scale_mean": 4.1,
  "integration_index": 0.78
}

Inclusion criteria

Both telemetry and think-aloud data captured for the same task with consistent integration signals.

Exclusion criteria

Tasks where AI is consciously bracketed as a separate tool (explicit pause-consult-resume pattern).

Agent Handshake [AHS]

Term code: AUG-0859 · Discipline: Agentic AI (AG) · Period: 1 · Language: de

Description

Initialer Kontaktmoment zwischen Mensch und autonomem Agenten. Faehigkeiten und Grenzen werden geklaert.

Operational definition

An Agent Handshake is recorded when two or more AI agents exchange a structured capability-declaration and task-acceptance message before commencing joint work, verified by presence of an explicit handshake-payload in the inter-agent message log.

Measurement schema

Protocol-conformance check via F1-score (precision-recall) on handshake-payload fields against a reference schema (capability-list, task-id, accept/reject, version). Reported per agent-pair with field-level recall.

Inclusion criteria

Structured handshake-payload present before joint-task initiation with F1 greater than 0.7.

Exclusion criteria

Ad-hoc agent-to-agent messaging without schema-conformant handshake.

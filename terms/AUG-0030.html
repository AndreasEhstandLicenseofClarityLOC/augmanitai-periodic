Contextual Gravity [COG]

Term code: AUG-0030 · Discipline: SY (SY) · Period: 2 · Language: de

Description

Mit wachsendem Kontext entwickelt eine KI-Sitzung eine natuerliche thematische Schwerkraft.

Operational definition

Contextual Gravity is observed when prior context-elements in a session (earlier messages, established persona, named entities) exert measurable pull on subsequent AI outputs such that later responses regress toward earlier framings even after explicit topic-shift. The effect is operationalised through within-session context-anchor probes inserted at fixed positions.

Measurement schema

Granger causality test between time-lagged context-anchor features (e.g., earlier persona-tokens) and downstream AI-output features (lexical, stylistic). Pull-strength reported as F-statistic with effect-size for each anchor-target pair across a sliding window.

Inclusion criteria

Context-anchor present in early turns with downstream AI-output measurably influenced (Granger p less than 0.05).

Exclusion criteria

Anchors that are explicitly re-cited by the user in later turns, removing the pull-signal.
